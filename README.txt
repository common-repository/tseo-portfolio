Plugin to display a gallery of websites developed with WordPress.

== Description ==

TSEO Portfolio allows you to display a gallery of websites that you have developed with WordPress. Each entry displays an image, title, description, and six buttons that link to the website, PageSpeed Mobile, PageSpeed Desktop, Pingdom, GTmetrix, and Google Rich Results, enabling each visitor to analyze the quality of each website.

**Key Features**:
- Display a visually appealing gallery of your web projects.
- Showcase your web development skills to your audience.
- Enable visitors to assess the quality of each website with ease.
- Seamlessly link to critical performance analysis tools like PageSpeed Mobile, PageSpeed Desktop, Pingdom, GTmetrix, and Google Rich Results.
- Enhance your online portfolio's impact and professionalism.

Whether you're a web developer, designer, or digital agency, TSEO Portfolio simplifies the process of presenting your work in a user-friendly and engaging manner. Elevate your online presence and impress your clients with the quality of your web projects today!

== Installation ==

1. Upload the tseoportfolio folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the new "TSEO Portfolio" menu in your dashboard to add and manage your websites.

== Frequently Asked Questions ==

= Does this plugin work on multisite? =
Yes, the plugin has been tested and verified on WordPress Multisite.

= Does this plugin optimize website resources? =
No. The goal of this plugin is to showcase the quality of your web development and design projects to your website visitors.

= How do I add websites to the portfolio gallery? =
You can add websites to the portfolio gallery by navigating to the "TSEO Portfolio > Add new" menu in your WordPress dashboard. From there, you can easily manage and add new websites to your portfolio.

= Can I customize the appearance of the portfolio gallery? =
Yes, you can customize the appearance of the portfolio gallery to match your website's design. You may explore customization options within the plugin's settings.

= Is technical support available for this plugin? =
Yes, technical support is available. If you encounter any issues or have questions, please reach out to our support team for assistance.

= Are there any limitations on the number of websites I can showcase with this plugin? =
There are no limitations on the number of websites you can showcase using this plugin. You can add as many websites as you like to your portfolio gallery.

= How can I analyze the performance of each website in the gallery? =
Each website in the gallery provides links to various analysis tools, such as PageSpeed Mobile, PageSpeed Desktop, Pingdom, GTmetrix, and Google Rich Results. Visitors can use these tools to assess the performance and quality of each website.

= How do I go about analyzing each link? =
This plugin is very straightforward and doesn't require any external APIs. Simply, in each field, you have an external link for analysis. Then, you copy the URL from the result and paste it into the corresponding field.

= Can I use this plugin for personal and commercial websites? =
Yes, you can use this plugin for both personal and commercial websites. It is a versatile tool suitable for showcasing various types of web projects.
